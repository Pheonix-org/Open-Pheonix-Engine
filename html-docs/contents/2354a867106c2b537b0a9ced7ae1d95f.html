formatUnsignedIntUTF16

UMLOperation
UML :: openJDK :: java :: lang :: Integer :: formatUnsignedIntUTF16

Description

Format an {@code int} (treated as unsigned) into a byte buffer (UTF16 version). If {@code len} exceeds the formatted ASCII representation of {@code val}, {@code buf} will be padded with leading zeroes. @param val the unsigned int to format @param shift the log2 of the base to format in (4 for hex, 3 for octal, 1 for binary) @param buf the byte buffer to write to @param len the number of characters to write
