Stringifiers

UMLClass
UML :: openJDK :: java :: lang :: invoke :: StringConcatFactory :: Stringifiers

Description

Public gateways to public "stringify" methods. These methods have the form String apply(T obj), and normally delegate to {@code String.valueOf}, depending on argument's type.
