writeObject

UMLOperation
UML :: openJDK :: java :: lang :: Throwable :: writeObject

Description

Write a {@code Throwable} object to a stream. A {@code null} stack trace field is represented in the serial form as a one-element array whose element is equal to {@code new StackTraceElement("", "", null, Integer.MIN_VALUE)}.
