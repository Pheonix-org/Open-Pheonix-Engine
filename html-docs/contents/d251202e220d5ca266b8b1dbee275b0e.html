Description

Compares two {@code Short} objects numerically. @param anotherShort the {@code Short} to be compared. @return the value {@code 0} if this {@code Short} is equal to the argument {@code Short}; a value less than {@code 0} if this {@code Short} is numerically less than the argument {@code Short}; and a value greater than {@code 0} if this {@code Short} is numerically greater than the argument {@code Short} (signed comparison). @since 1.2
