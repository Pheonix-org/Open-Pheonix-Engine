setDisplayMultiplyer

UMLOperation
UML :: opex :: frontend :: window :: prefabs :: Splash :: setDisplayMultiplyer

Description

Sets the multiplier for minimum display length. Multiplier is equal to the minimum display time in seconds; i.e multiplier 1L results 1 second minimum display time. minimum display time = (1000L multiplier) Once minimum display time has passed, the splash will close the instant the engine declares sucessfuls startup. If the engine completed before the splash has closed, it will wait for the splash to close before executing post startup and client payloads. @param multiplyer
