setPriority

UMLOperation
UML :: openJDK :: java :: lang :: Thread :: setPriority

Description

Changes the priority of this thread. <p> First the {@code checkAccess} method of this thread is called with no arguments. This may result in throwing a {@code SecurityException}. <p> Otherwise, the priority of this thread is set to the smaller of the specified {@code newPriority} and the maximum permitted priority of the thread's thread group. @param newPriority priority to set this thread to @throws IllegalArgumentException If the priority is not in the range {@code MINPRIORITY} to {@code MAXPRIORITY}. @throws SecurityException if the current thread cannot modify this thread. @see #getPriority @see #checkAccess() @see #getThreadGroup() @see #MAXPRIORITY @see #MINPRIORITY @see ThreadGroup#getMaxPriority()
