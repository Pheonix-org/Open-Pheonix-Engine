Description

Compares two {@code int} values numerically. The value returned is identical to what would be returned by: <pre> Integer.valueOf(x).compareTo(Integer.valueOf(y)) </pre> @param x the first {@code int} to compare @param y the second {@code int} to compare @return the value {@code 0} if {@code x == y}; a value less than {@code 0} if {@code x < y}; and a value greater than {@code 0} if {@code x > y} @since 1.7
