poll

UMLOperation
UML :: openJDK :: java :: lang :: ref :: ReferenceQueue :: poll

Description

Must hold lock Polls this queue to see if a reference object is available. If one is available without further delay then it is removed from the queue and returned. Otherwise this method immediately returns {@code null}. @return A reference object, if one was immediately available, otherwise {@code null}
