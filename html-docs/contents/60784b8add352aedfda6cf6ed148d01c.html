highestOneBit

UMLOperation
UML :: openJDK :: java :: lang :: Long :: highestOneBit

Description

Returns a {@code long} value with at most a single one-bit, in the position of the highest-order ("leftmost") one-bit in the specified {@code long} value. Returns zero if the specified value has no one-bits in its two's complement binary representation, that is, if it is equal to zero. @param i the value whose highest one bit is to be computed @return a {@code long} value with a single one-bit, in the position of the highest-order one-bit in the specified value, or zero if the specified value is itself equal to zero. @since 1.5
