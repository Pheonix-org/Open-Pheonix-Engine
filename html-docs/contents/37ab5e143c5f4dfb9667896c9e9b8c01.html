CJARMeta

UMLClass
UML :: opex :: backend :: io :: cjar :: CJARMeta

Description

Serializable meta data container for CJAR's.

A filled instance is serialized and stored within a cjar.

Contains file expectency, signing, and compatability information about the cjar.

Attributes

Visibility Name Type Description
public minimumOPEXVersion Version

Version of the cjar package standard.
public minimumGameVersion Version
public mainClassName String

Name of the main class which this package is intended for, i.e the game a dlc is intended for. Name must be simple class name used in the OPEX engine, of the class which extends OPEXGame.
