Class to track and run user level shutdown hooks registered through {@link Runtime#addShutdownHook Runtime.addShutdownHook}. @see java.lang.Runtime#addShutdownHook @see java.lang.Runtime#removeShutdownHook

Attributes

Visibility Name Type Description
private hooks IdentityHashMap

The set of registered hooks

Operations

Visibility Name Description
private ApplicationShutdownHooks()
package add(hook: Thread): void

Add a new shutdown hook. Checks the shutdown state and the hook itself, but does not do any security checks.
package remove(hook: Thread): boolean

Remove a previously-registered hook. Like the add method, this method does not do any security checks.
package runHooks(): void

Iterates over all application hooks creating a new thread for each to run in. Hooks are run concurrently and this method waits for them to finish.
