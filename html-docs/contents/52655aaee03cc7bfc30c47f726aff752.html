UnicodeBlock

UMLOperation «constructor»
UML :: openJDK :: java :: lang :: Character :: UnicodeBlock :: «constructor»UnicodeBlock

Description

Creates a UnicodeBlock with the given identifier name. This name must be the same as the block identifier.
