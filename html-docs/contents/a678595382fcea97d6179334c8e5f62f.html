code_ldc

UMLOperation
UML :: openJDK :: java :: lang :: reflect :: ProxyGenerator_v49 :: code_ldc

Description

Use the "wide" instruction modifier for local variable indexes that do not fit into an unsigned byte. Generate code for an "ldc" instruction for the given constant pool index (the "ldc_w" instruction is used if the index does not fit into an unsigned byte). The code is written to the supplied stream.
