Description

Constructs a new exception with the specified detail message and cause. <p>Note that the detail message associated with {@code cause} is <i>not</i> automatically incorporated in this exception's detail message. @param message the detail message (which is saved for later retrieval by the {@link Throwable#getMessage()} method). @param cause the cause (which is saved for later retrieval by the {@link Throwable#getCause()} method). (A {@code null} value is permitted, and indicates that the cause is nonexistent or unknown.) @since 1.5
