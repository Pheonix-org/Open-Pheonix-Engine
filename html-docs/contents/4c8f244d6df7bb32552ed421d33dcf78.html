UML :: opex :: backend :: runtime :: environment :: OPEX

Description

Main executable entry point for starting OPEX. Is instantiable, but is static. Creating an instance of this class is the intended entry point; @example new OPEX(OPEXGame game); @since 2020.4.20.A @author Jordan Gray
