Predefined values of <a href="package-summary.html#nominal">nominal descriptor</a> for common constants, including descriptors for primitive class types and other common platform types, and descriptors for method handles for standard bootstrap methods. @see ConstantDesc @since 12

Attributes

Visibility Name Type Description
public DEFAULT_NAME String

Invocation name to use when no name is needed, such as the name of a constructor, or the invocation name of a dynamic constant or dynamic callsite when the bootstrap is known to ignore the invocation name.
