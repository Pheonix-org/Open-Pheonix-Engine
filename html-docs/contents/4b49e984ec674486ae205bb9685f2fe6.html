getOwnerType

UMLOperation
UML :: openJDK :: java :: lang :: reflect :: ParameterizedType :: getOwnerType

Description

Returns a {@code Type} object representing the type that this type is a member of. For example, if this type is {@code O<T>.I<S>}, return a representation of {@code O<T>}. <p>If this type is a top-level type, {@code null} is returned. @return a {@code Type} object representing the type that this type is a member of. If this type is a top-level type, {@code null} is returned @throws TypeNotPresentException if the owner type refers to a non-existent type declaration @throws MalformedParameterizedTypeException if the owner type refers to a parameterized type that cannot be instantiated for any reason @since 1.5
