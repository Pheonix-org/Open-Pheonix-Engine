PrimitiveTypeInfo

UMLClass
UML :: openJDK :: java :: lang :: reflect :: ProxyGenerator_v49 :: PrimitiveTypeInfo

Description

If Throwable is declared to be thrown by the proxy method, then no catch blocks are necessary, because the invoke can, at most, throw Throwable anyway. Ignore types that cannot be thrown by the invoke method. Compare this exception against the current list of exceptions that need to be caught: if a superclass of this exception is already on the list to catch, then ignore this one and continue; if a subclass of this exception is on the list to catch, then remove it; A PrimitiveTypeInfo object contains assorted information about a primitive type in its public fields. The struct for a particular primitive type can be obtained using the static "get" method.
