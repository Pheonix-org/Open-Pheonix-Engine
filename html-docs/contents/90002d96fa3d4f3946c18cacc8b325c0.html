getExponent

UMLOperation
UML :: openJDK :: java :: lang :: Math :: getExponent

Description

Bitwise convert f to integer, mask out exponent bits, shift to the right and then subtract out float's bias adjust to get true exponent value Returns the unbiased exponent used in the representation of a {@code double}. Special cases: <ul> <li>If the argument is NaN or infinite, then the result is {@link Double#MAXEXPONENT} + 1. <li>If the argument is zero or subnormal, then the result is {@link Double#MINEXPONENT} -1. </ul> @param d a {@code double} value @return the unbiased exponent of the argument @since 1.6
