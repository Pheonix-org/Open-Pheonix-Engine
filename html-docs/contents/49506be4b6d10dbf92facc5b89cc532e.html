Cleanable

UMLInterface
UML :: openJDK :: java :: lang :: ref :: Cleaner :: Cleanable

Description

{@code Cleanable} represents an object and a cleaning action registered in a {@code Cleaner}. @since 9

Operations

Visibility Name Description
package clean(): void

Unregisters the cleanable and invokes the cleaning action. The cleanable's cleaning action is invoked at most once regardless of the number of calls to {@code clean}.
