StackStreamFactory class provides static factory methods to get different kinds of stack walker/traverser. AbstractStackWalker provides the basic stack walking support fetching stack frames from VM in batches. AbstractStackWalker subclass is specialized for a specific kind of stack traversal to avoid overhead of Stream/Lambda 1. Support traversing Stream<StackFrame> 2. StackWalker::getCallerClass 3. AccessControlContext getting ProtectionDomain
