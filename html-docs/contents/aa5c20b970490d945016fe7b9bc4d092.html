log10

UMLOperation
UML :: openJDK :: java :: lang :: StrictMath :: log10

Description

Returns the base 10 logarithm of a {@code double} value. Special cases: <ul><li>If the argument is NaN or less than zero, then the result is NaN. <li>If the argument is positive infinity, then the result is positive infinity. <li>If the argument is positive zero or negative zero, then the result is negative infinity. <li> If the argument is equal to 10<sup><i>n</i></sup> for integer <i>n</i>, then the result is <i>n</i>. </ul> @param a a value @return the base 10 logarithm of {@code a}. @since 1.5
