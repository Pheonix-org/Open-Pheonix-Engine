UMLOperation
UML :: openJDK :: java :: lang :: reflect :: Parameter :: getName

Description

Returns the name of the parameter. If the parameter's name is {@linkplain #isNamePresent() present}, then this method returns the name provided by the class file. Otherwise, this method synthesizes a name of the form argN, where N is the index of the parameter in the descriptor of the method which declares the parameter. @return The name of the parameter, either provided by the class file or synthesized if the class file does not provide a name.
