Description

Registers an object and a cleaning action to run when the object becomes phantom reachable. Refer to the <a href="#compatible-cleaners">API Note</a> above for cautions about the behavior of cleaning actions. @param obj the object to monitor @param action a {@code Runnable} to invoke when the object becomes phantom reachable @return a {@code Cleanable} instance
